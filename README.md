## Sustainable Home Improvements Recommendations

Built for myself, honestly.

FastAPI + Vue + TS + deck.gl SPA.

Intended to recommend green tech (solar panels, heat pumps, insulation etc.) based on location & property features.

For example - South-East Scotland is pretty good for solar, due to decent sunlight and non-problematic temperatures for panel overheating.

Unversioned pre-alpha.

AI-assisted for scaffolding and boilerplate. Hand-architected and carefully constructed.
